Hi 👋 My name is Tarun Yadav
============================

Web Developer & AI Engineer
---------------------------

I am a web developer and AI enthusiast who is very passionate and dedicated to making modern, responsive web applications that use the latest AI technologies. By being good in both modern frontend frameworks and machine learning, I develop projects that have nice appealing user interfaces along with robust backend intelligence.

My Projects:

🤖 barista-ai - An AI-driven advanced portfolio management with risk analysis

and investment advice provided by agents

🎵 Bragi - A Next.js and TypeScript built music player that is modern and stylish,

proving the use of up-to-date web development techniques

Technical Stack:

• Frontend: TypeScript, React, Next.js

• Backend: Python, AI/ML frameworks

• Full Stack: Web APIs, LLM integration, modern tooling

Philosophy:

The combination of artistic design with smart functionality is where I place my faith.

Be it making web interfaces that work well with all types of devices or training AI models

for the purpose of solving real-world problems, I will always be there with my solutions

that are tasteful and efficient.

Apart from coding, I am interested in:

✨ Discovering the use of AI and machine learning in web development

✨ More open-source project participation

💡 Networking with the developer community
